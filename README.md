## ✨ Aperçu

Portfolio personnel conçu et développé en HTML/CSS/JS pur — **zéro framework, zéro dépendance externe**, uniquement des Google Fonts en CDN. Le site est un fichier unique (`index.html`) prêt à être déployé sur n'importe quelle plateforme.

### 🎨 Design

- **Style** : Editorial Dark · Soft Brutalism Millennial
- **Typographie** : Playfair Display (titres) + IBM Plex Sans (corps) + IBM Plex Mono (code/labels)
- **Palette** : Dark slate `#090910` · Amber doré `#e8a838` · Crème `#ece9e0`



---

## 🚀 Fonctionnalités

| Fonctionnalité | Description |
|---|---|
| 🌙☀️ **Dark / Light Mode** | Toggle en navbar, préférence sauvegardée en `localStorage` |
| ⌨️ **Typewriter Effect** | Rotation des rôles dans le hero (Dev Web, Électrotechnicien, IoT…) |
| 🖱️ **Curseur personnalisé** | Curseur amber avec anneau à inertie |
| 📊 **Compteurs animés** | Stats du hero qui s'incrémentent à l'entrée dans le viewport |
| 📈 **Barres de compétences** | Animation au scroll via IntersectionObserver |
| 🔍 **Projets filtrables** | Filtres par catégorie (Web App, IoT, API, Médias) |
| 🌐 **Topology SVG** | Décoration géométrique animée en fond du hero |
| 💬 **Section Témoignages** | Cards avec citations des collaborateurs |
| 📬 **Formulaire de contact** | Redirige vers WhatsApp avec les infos pré-remplies |
| 📱 **Responsive** | Adapté mobile / tablette / desktop (375 → 1440px) |
| ✨ **Scroll Reveal** | Animations d'entrée au scroll sur toutes les sections |

## 🗂️ Structure du site

```
index.html
│
├── 🦸 Hero          — Photo, nom, typewriter, stats, CTAs
├── 🛠️ Services      — 6 cartes de services + logos partenaires
├── 💡 Compétences   — Grille 12 skills avec barres animées
├── 📁 Projets       — 6 projets filtrables (DREAM XR, Sérénité, RHX, TaskTrek…)
├── 💬 Témoignages   — 3 témoignages de collaborateurs
├── 📱 Médias        — Techkick & Woman of Excellence (pages Facebook)
├── 👤 Parcours      — Bio, timeline, certifications, download CV
├── 📬 Contact       — Formulaire + canaux (WhatsApp, Email, Localisation)
└── 🦶 Footer        — Réseaux sociaux, copyright
```

---

## 🧰 Stack technique

```
HTML5       — Structure sémantique, un seul fichier
CSS3        — Variables CSS, Grid, Flexbox, animations, transitions
JavaScript  — Vanilla JS, IntersectionObserver, LocalStorage
Fonts       — Google Fonts CDN (Playfair Display, IBM Plex Sans/Mono)
Images      — Photo de profil encodée en base64 inline
```

Aucune dépendance npm. Aucun build step. Aucun framework.
